17本流派
TH17

根蔓武神

特点

有一定容错，适应性高，适合传奇联赛使用

核心技巧

隐形神盾前戏断两边

  • 兵种配置：5天使 + 5根蔓 + 8武神 + 泰坦，冰人，超炸x1
  • 兵种配置：6天使 + 10飞龙 + 守卫兵
  • 法术配置：9隐形 + 1召回
  • 攻城机器：攻城营
  • 援军：超猪 + 锅炉 + 根蔓法术 + 冰障法术

操作步骤

  • 神盾前戏断一边后召回
  • 下回神盾开飞矛+工程营，在断一边
  • 主力进阵
  • 根蔓法术抓部分重火力，分伤害
  • 冰障法术抓时机保前排
撰写：燧 | 适用阵型：大部分
13本流派
TH13

天女龙流

流派特点

天女配合飞龙，操作简单容错率高。

核心技巧

开局神盾前戏优先破坏火力猛的区域后召回，女王巨剑两点一线清空气炮后下主力。

  • 兵种配置：6天使 + 10飞龙 + 守卫兵
  • 法术配置：1狂暴 + 6隐形 + 3地震
  • 攻城机器：攻城飞艇
  • 援军：9球 + 2隐形

操作步骤

  • 开局神盾前戏摧毁关键防御区
  • 召回神盾
  • 天女切入
  • 女王使用巨剑技能清空气炮
  • 主力飞龙进攻
撰写：阳光总在风雨后

隐闰龙流

流派特点

前戏技术要求偏高，容错低，适用阵型多

核心技巧

神盾隐形前戏并断边，双王再断一边

  • 兵种配置：14飞龙 + 收尾兵
  • 法术配置：11隐形
  • 攻城机器：攻城气球
  • 援军：2飞龙 + 2隐形法术

操作步骤

  • 神盾前戏摧毁关键防御顺带断边
  • 神盾前戏结束后下双王断边
  • 主力进阵
  • 坐等三星
撰写：燧
12本流派
TH12

天女矿猪

流派特点

天女前戏配合矿工野猪主力，多线推进，适合各种阵型。

核心技巧

将阵型拆分为田字格，选择包含援军城堡和天鹰的区块进行突破。

  • 兵种配置：5天使 + 10矿工 + 15野猪
  • 法术配置：3治疗 + 2狂暴 + 1毒药
  • 攻城机器：攻城训练营
  • 援军：野猪骑士 + 冰冻法术

操作步骤

  • 将敌方阵型拆分为四个等分正方形
  • 选择包含援军城堡和天鹰的区块
  • 左右小断边清理外围建筑
  • 天女切入清除援军和天鹰火炮
  • 阵型变为三个相连正方形(L型路线)
  • 左右大断边后下主力(先矿工后野猪)
  • 精准使用治疗法术覆盖主力部队
撰写：燧
